# Hardware Wallet Support

RenVault now supports hardware wallets for enhanced security.

## Supported Devices

- **Ledger**: Connect via USB and use the Stacks app.
- **Trezor**: Connect via web interface or bridge.

## Setup

1. Install dependencies: `npm install`
2. For Ledger: Ensure Stacks app is installed on your device.
3. For Trezor: Use Trezor Bridge or web interface.

## Usage

Select the hardware wallet provider in the app, follow connection instructions, and sign transactions securely on the device.

## Security

Transactions are signed on the hardware device, never exposing private keys to the application.
