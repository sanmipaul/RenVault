# Transaction History

RenVault now provides comprehensive transaction history viewing capabilities.

## Features

- **Complete History**: View all transactions for your wallet address
- **Filtering**: Filter by transaction type (sent, received, contract calls)
- **Sorting**: Sort by date or amount in ascending/descending order
- **Pagination**: Navigate through large transaction lists
- **Explorer Integration**: Direct links to Stacks Explorer for detailed views

## Usage

1. Connect your wallet
2. Navigate to the Transaction History section
3. Use filters and sorting options as needed
4. Click "View" links to see transaction details on Stacks Explorer

## API Integration

Uses the Stacks Blockchain API to fetch transaction data securely.
